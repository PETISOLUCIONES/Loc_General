Purchase order by project

OVERVIEW

This module allows you to create a purchase request from a project.
The module also offers the possibility of having the project analytical account at the level of the purchase request lines.

Features

Add a button to create a purchase request.
Automatically assign the analytical account on the purchase lines.

Screenshots

Step 1: Click on the Create a supplier order button

Step 2: Start filling out the purchase order form

You can see that the analytical account is automatically passed to the level of the purchase lines
