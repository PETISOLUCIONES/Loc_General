This module helps the user to enable the records of customer and vendor same as older versions of Odoo.

User needs to update the boolean for customer and vendor from the contacts view.

Once user will update the customer boolean then the sale order will display the records of only enabled customers.

Once user will update the vendor boolean then the purchase order will display the records of only enabled vendors.
