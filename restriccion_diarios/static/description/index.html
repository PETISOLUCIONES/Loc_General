Habilitar grupo de seguridad para asignar diarios.


El administrador puede restringir usuarios dentro de los diarios para que solo esos usuarios puedan acceder a esos diarios. Si activa el grupo de seguridad, se mostrará el campo "Diarios".


El administrador también puede definir diarios restringidos dentro de usuarios para que solo ese usuario pueda acceder a esos diarios. Si activa el grupo de seguridad, se mostrará el campo "Usuarios".


El administrador tiene derechos de acceso en configuración, por lo que puede ver todos los diarios.
