# Restricción de Diarios por Usuario

## Acerca de
Este módulo restringe diarios para usuarios específicos. Puede agregar usuarios con acceso en la configuración del diario, solo los usuarios permitidos pueden acceder a ese diario. Los usuarios se asignan a diarios específicos como facturas, facturas de proveedor, efectivo, banco, ventas y compras. Los usuarios no pueden acceder a un diario donde el diario no esté disponible para ese usuario.

## Instalación
1. Copiar archivos del módulo a la carpeta de addons
2. Reiniciar servicio de Odoo (sudo service odoo-server restart)
3. Ir a la instancia de Odoo y abrir Apps (asegúrese de activar modo debug)
4. Hacer clic en "Actualizar lista de aplicaciones"
5. Buscar el nombre del módulo y hacer clic en instalar
